# कर्मIO 

**कर्मIO** is a web-based Employee Task Management System designed to streamline task allocation, tracking, and management for organizations. It helps managers and employees efficiently manage daily tasks, monitor progress, and ensure smooth workflow within teams.

## Features

* **Task Management:** Create, assign, and track tasks for employees.
* **User Roles:** Separate access for Admin and Employee roles.
* **Task Status Tracking:** Track task progress with status updates (Pending, In Progress, Completed).
* **Notifications:** Get alerts/updates for new and ongoing tasks.
* **Responsive Design:** Accessible on both desktop and mobile devices.
* **Employee Dashboard:** View assigned tasks, deadlines, and status updates in one place.

---

## Technology Stack

* **Frontend:** React.js, Bootstrap
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Firebase
* **APIs:** RESTful APIs for task and user management
* **Tools & Platforms:** GitHub, VS Code, Postman, Cloudinary

## Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/karmio.git
```

2. Navigate to the project directory:

```bash
cd karmio
```

3. Install dependencies for frontend and backend:

```bash
npm install
```

. Run the project:

```bash
# For backend
npm start  

# For frontend
npm run dev
```
